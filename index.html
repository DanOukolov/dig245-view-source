Step 1: How to See the SRC and Use Tools

Favorite Browser: Duckduckgo
  1. Right click anywhere on the page
  2. Click Inspect
  3. Check the elements tab and styles tab
  4. You did it! You are now developing!
    What you can do with Dev Tools
  • One: You can inspect all of the elements made up in the HTML structure as well as the styles used in the style sheets. You can click on one of these lines of codes and you have a plethora of tools you can use from there(example below). You can also just type things directly into the elements tab.
  • Two: You can hover over a line of code, see where it is on the page, right click on it, and copy the selector. After copying the selector, can then paste the selector into the console and add styling to the element.
  • Three: You can select and right click an element on the website and hit inspect. From here, the elements tab shows the HTML and the styles tab will show all of the styles associated with the element specfic to that element alone. This is useful because it narrows the search down for specific styling codes in the stylesheet.
  • Four: Overall, this whole process is called "Stripping the Sheets." I call it this because we can use the SRC to strip the sylesheets off of the website's style tab.
